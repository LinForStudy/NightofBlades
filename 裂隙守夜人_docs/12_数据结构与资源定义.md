# 数据结构与资源定义

## 1. SkillData

建议字段：

```gdscript
class_name SkillData
extends Resource

@export var skill_id: StringName
@export var display_name: String
@export_multiline var description: String
@export var icon: Texture2D
@export var skill_scene: PackedScene
@export var base_cooldown: float
@export var max_level: int = 5
@export var tags: Array[StringName]
@export var upgrades: Array[SkillUpgradeData]
```

## 2. SkillUpgradeData

```gdscript
class_name SkillUpgradeData
extends Resource

@export var upgrade_id: StringName
@export var skill_id: StringName
@export var target_level: int
@export var branch_id: StringName
@export var rarity: int
@export var prerequisites: Array[StringName]
@export_multiline var description: String
@export var modifiers: Dictionary
```

## 3. EnemyData

```gdscript
class_name EnemyData
extends Resource

@export var enemy_id: StringName
@export var display_name: String
@export var enemy_scene: PackedScene
@export var max_health: float
@export var move_speed: float
@export var base_damage: float
@export var armor: float
@export var experience_value: int
@export var tags: Array[StringName]
```

## 4. WaveData

```gdscript
class_name WaveData
extends Resource

@export var start_time: float
@export var end_time: float
@export var spawn_interval: float
@export var max_alive: int
@export var enemy_entries: Array[EnemySpawnEntry]
@export var elite_chance: float
@export var health_multiplier: float = 1.0
@export var damage_multiplier: float = 1.0
```

## 5. UpgradeOption

运行时对象：

```gdscript
class_name UpgradeOption
extends RefCounted

var upgrade_id: StringName
var title: String
var description: String
var icon: Texture2D
var rarity: int
var apply_callback: Callable
```

## 6. PlayerRunState

保存单局运行状态：

```gdscript
class_name PlayerRunState
extends RefCounted

var level: int = 1
var current_exp: int = 0
var max_health: float = 100.0
var attack_multiplier: float = 1.0
var cooldown_multiplier: float = 1.0
var active_skills: Dictionary = {}
var passive_stacks: Dictionary = {}
var selected_branches: Dictionary = {}
var reroll_count: int = 1
```

## 7. BattleStats

需要统计：

- damage_dealt
- damage_taken
- kills
- elites_killed
- bosses_killed
- highest_combo
- skill_damage
- perfect_dodges
- healing_done
- run_duration

## 8. 运行时数据原则

- 静态配置保存在 Resource
- 单局变化保存在 RunState
- 跨局变化保存在 SaveData
- 不允许把局内等级直接写入 SkillData Resource
