# 技能系统与技能表

## 1. 技能框架

所有主动技能继承统一基类或统一组件接口。

每个技能包含：

- skill_id
- display_name
- description
- icon
- cooldown
- cast_time
- recovery_time
- damage
- damage_type
- tags
- max_level
- current_level
- evolution_branch
- scene
- effect_scene
- audio
- upgrade_pool

## 2. 技能生命周期

技能执行流程：

1. 检查冷却
2. 检查玩家状态
3. 锁定或计算方向
4. 进入施法状态
5. 播放动画
6. 在指定帧生成效果
7. 技能结束
8. 开始冷却
9. 发出技能释放信号

## 3. 冷却系统

- 冷却由统一 SkillCooldownComponent 管理
- UI 通过信号监听剩余冷却
- 冷却缩减统一使用倍率
- 最低冷却不得低于基础值的20%
- 大招不使用普通冷却，而使用能量

## 4. MVP 主动技能

## 4.1 烈焰剑气

基础：

- 冷却：5秒
- 基础伤害：30
- 穿透：3
- 附加燃烧：4秒
- 燃烧每秒伤害：5

等级：

- Lv1：解锁
- Lv2：伤害+25%，穿透+2
- Lv3A 炎爆：燃烧敌人死亡时爆炸
- Lv3B 巨刃：剑气宽度+100%，无限穿透，速度降低
- Lv4A：爆炸范围+35%，爆炸伤害+30%
- Lv4B：巨刃伤害+45%，击退增强
- Lv5A 炼狱连爆：爆炸可点燃附近敌人
- Lv5B 焚天巨刃：释放超大剑气并留下燃烧路径

## 4.2 雷影冲刺

基础：

- 冷却：7秒
- 向目标方向冲刺
- 冲刺过程无敌
- 路径伤害：25

等级：

- Lv1：解锁
- Lv2：距离+20%，冷却-10%
- Lv3A 连锁：命中后向附近敌人释放闪电
- Lv3B 残影：终点留下攻击残影
- Lv4A：连锁次数+2
- Lv4B：残影持续时间+50%
- Lv5A 万雷穿行：连续锁定三次冲刺
- Lv5B 雷影军团：生成两个残影并复制下一次平A

## 4.3 旋刃风暴

基础：

- 冷却：8秒
- 持续时间：2秒
- 每0.2秒造成一次伤害
- 释放期间可移动，移动速度降低20%

等级：

- Lv1：解锁
- Lv2：范围+20%，持续时间+0.5秒
- Lv3A 吸附：吸引普通敌人
- Lv3B 飞刃：周期性向四周发射飞刃
- Lv4A：吸力增强，伤害+20%
- Lv4B：飞刃数量+4
- Lv5A 吞噬旋涡：结束时造成中心爆发
- Lv5B 无尽刀阵：技能结束后飞刃继续存在3秒

## 4.4 影分身

后续版本技能，但框架需要支持：

- 召唤实体
- 分身继承部分玩家属性
- 分身复制普通攻击
- 分身有持续时间
- 分身不能无限递归召唤分身

## 4.5 冰封领域

后续版本技能，但框架需要支持：

- 地面区域
- 持续效果
- 减速
- 冻结累计
- 区域结束爆发

## 5. 大招：天陨斩

基础流程：

1. 玩家无敌
2. 镜头轻微拉近
3. 玩家跃起
4. 短暂停顿
5. 向下高速斩击
6. 产生大范围冲击波
7. 清除普通敌人的弱弹幕
8. 对 Boss 造成高伤害
9. 恢复正常时间与摄像机

基础伤害：200。

大招不应该直接秒杀 Boss，但应当明显削减血量。

## 6. 元素系统

MVP 支持：

- Physical
- Fire
- Lightning
- Ice
- Shadow

状态效果：

- Burning：持续伤害
- Shocked：更容易触发连锁雷电
- Frozen：无法行动
- Vulnerable：受到伤害增加
- Bleeding：移动时受到额外伤害

MVP 可先实现 Burning 和 Frozen，其他保留接口。

## 7. 技能协同

技能可以通过标签产生联动。

例：

- 火焰技能命中冰冻敌人：触发融爆
- 雷电命中燃烧敌人：产生爆燃电弧
- 分身释放带有 `copyable` 标签的技能
- 旋刃命中带有燃烧状态的敌人时提高爆炸概率

不要在技能代码中写死对另一个技能名称的判断，应使用标签和状态系统。

## 8. 验收标准

- 技能冷却稳定
- 技能升级后参数立即生效
- 选择分支后不会出现另一分支升级
- 技能达到5级后不再出现在升级池
- 同一技能的视觉、范围和行为会随着升级明显变化
