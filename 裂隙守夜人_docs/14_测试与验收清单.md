# 测试与验收清单

## 1. 玩家控制

- [ ] 移动加速和减速正常
- [ ] 左右朝向正确
- [ ] 跳跃高度一致
- [ ] 土狼时间有效
- [ ] 跳跃缓冲有效
- [ ] 穿透平台正常
- [ ] 摄像机不抖动

## 2. 战斗

- [ ] 三段连击顺序正确
- [ ] 连击超时后重置
- [ ] 攻击方向正确
- [ ] Hitbox 只在有效帧开启
- [ ] 同一目标不会异常多次受击
- [ ] 击退方向正确
- [ ] 受击无敌有效
- [ ] 玩家死亡后无法继续输入
- [ ] 命中停顿不会永久暂停游戏

## 3. 闪避

- [ ] 闪避有冷却
- [ ] 闪避期间无敌
- [ ] 完美闪避窗口准确
- [ ] 完美闪避只触发一次
- [ ] 闪避结束后状态正确

## 4. 敌人

- [ ] 近战敌人可以接近并攻击
- [ ] 远程敌人保持距离
- [ ] 自爆怪有预警
- [ ] 飞行敌人不会掉到地面
- [ ] 敌人死亡后不继续攻击
- [ ] 精英词条正确生效

## 5. 波次

- [ ] 敌人按时间生成
- [ ] 最大存活数限制有效
- [ ] 不在玩家附近生成
- [ ] 波次倍率正确
- [ ] 精英出现概率正确
- [ ] Boss 出现后普通生成逻辑符合设计

## 6. 升级

- [ ] 经验正确累计
- [ ] 连续升级不会丢失
- [ ] 升级时世界暂停
- [ ] 候选不重复
- [ ] 满级技能不出现
- [ ] 分支互斥
- [ ] 重抽次数正确
- [ ] 升级效果立即生效

## 7. 技能

- [ ] 冷却正确
- [ ] UI 冷却遮罩正确
- [ ] 技能伤害正确
- [ ] 技能打断规则正确
- [ ] 技能分支表现不同
- [ ] 最终进化只出现一次
- [ ] 技能切换场景后重置

## 8. Boss

- [ ] Boss 登场演出只播放一次
- [ ] Boss 血条正确
- [ ] 阶段切换只触发一次
- [ ] Boss 攻击均有预警
- [ ] Boss 死亡后停止攻击
- [ ] 胜利流程正确

## 9. UI

- [ ] 1280×720 显示完整
- [ ] 技能图标正确
- [ ] 升级卡片文字不溢出
- [ ] 键盘和鼠标均可操作
- [ ] 暂停和恢复正常
- [ ] 结算数据正确

## 10. 性能

- [ ] 40个敌人稳定60FPS
- [ ] 80个敌人仍可流畅操作
- [ ] 大量经验球不会严重卡顿
- [ ] 大量伤害数字不会持续增长节点数量
- [ ] 对象池回收正确
- [ ] 长时间运行无明显内存增长

## 11. 回归要求

Codex 每次修改后必须至少检查：

- 项目能否启动
- 当前主场景是否运行
- 是否出现脚本解析错误
- 是否破坏玩家移动
- 是否破坏攻击输入
- 是否破坏场景路径
