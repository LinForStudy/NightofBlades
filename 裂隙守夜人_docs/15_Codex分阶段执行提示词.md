# Codex 分阶段执行提示词

## 总提示词

你正在实现一个 Godot 4.x 2D 横版像素动作 Roguelite 游戏《裂隙守夜人》。

必须遵守：

1. 先阅读项目内 docs 目录的相关文档。
2. 每次只完成当前阶段，不擅自扩展下一阶段。
3. 优先修复现有代码，不重复创建功能相同的脚本。
4. 所有新系统模块化、数据驱动。
5. 不把所有逻辑堆入 Player 或 BattleScene。
6. 修改前先检查现有目录、场景和脚本。
7. 保持 Godot 4.x GDScript 语法正确。
8. 完成后运行项目和检查错误。
9. 输出修改文件、实现内容、测试结果和剩余风险。
10. 如果正式素材缺失，使用明确的占位素材，不阻塞功能。
11. Windows 沙箱下如果补丁工具无法挂载可写根目录，直接使用同一工作区内的 PowerShell UTF-8 写入方式进行等价的最小文件修改，不重复尝试失败方式。
12. 不得宣称已经完成未运行验证的功能。
